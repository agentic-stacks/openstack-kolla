# Decision Guides

When an operator is setting up a new deployment, they need to make two key decisions:
networking mode and storage backend. These guides help you recommend the right choice.

| Decision | Guide | Options |
|---|---|---|
| Networking plugin | [networking.md](networking.md) | OVN, Open vSwitch, LinuxBridge |
| Storage backend | [storage.md](storage.md) | Ceph, LVM, NFS |
